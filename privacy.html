Privacy Policy for Bumper

Last Updated: November 2025

Bumper (“the Bot”) is a Discord bot created to automatically send scheduled messages. The Bot does not collect, store, or process any user data.

1. Data Collection

Bumper does not collect any of the following:

  • User IDs
  • Server IDs
  • Message content
  • Message logs
  • Any personal or identifiable information

2. Use of Data

Since Bumper does not collect any data, there is no data usage.

3. Data Storage

The Bot stores no data of any kind.

4. Data Deletion

No data is stored, so no deletion is required. For concerns, contact the developer below.
